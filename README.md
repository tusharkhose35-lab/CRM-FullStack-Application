# 🚀 CRM Full Stack Application

A modern Customer Relationship Management (CRM) system developed using Java Spring Boot and React.

## 📌 Features

- 🔐 User Authentication & Authorization
- 👨‍💼 Admin Dashboard
- 👨‍💻 User Dashboard
- 👤 Customer Dashboard
- 📋 Lead Management
- 🎫 Ticket Management
- 👥 Customer Management
- 💬 Feedback Management
- 📧 Email Integration
- 🔒 Spring Security
- 🎨 Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Bootstrap
- CSS3

### Backend
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- Maven

### Database
- MySQL
